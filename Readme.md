## 📁 Como baixar o projeto

```bash
# Clonar o repositório
$ git clone https://github.com/ElielAlves42/Chale-Hotel

# Entrar no diretório
$ cd Chale-Hotel

# instalar as dependências
$ yarn install

# Iniciar o projeto
$ yarn start
```
